# AI NPC - Next.js Project

A simple Next.js project with a single page displaying an iframe embedding the AI Avatar Closed game from itch.io.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
npm start
```
